# Ollama 全中文入门教程

> 适用版本：以上游 Release 为准。本文是首版教程草稿，正式操作前请同时核对上游文档。最后整理：2026-07-02。

## 开始之前

希望数据留在本地、愿意承担模型存储和硬件成本的AI用户与开发者。

支持平台：Windows、macOS、Linux。

## 推荐入门流程

1. 从官网选择对应系统安装包并完成安装
2. 打开终端运行 `ollama run qwen3:4b`
3. 等待模型下载完成后输入第一个问题
4. 运行 `ollama list` 检查本地模型
5. 不再需要时用 `ollama rm 模型名` 清理空间

## 第一次使用的验证标准

- 软件或服务能够正常启动。
- 核心功能使用测试数据完成一次闭环。
- 知道配置、数据和日志保存在哪里。
- 知道如何备份、升级和卸载。
- 涉及重要数据时，已经完成恢复测试。

## 常见误区

- 使用体验高度依赖内存、显存和模型大小
- 模型文件通常占用数GB到数十GB空间
- 本地运行不等于模型输出天然可靠

## 安全与隐私

开源不自动等于安全。安装前应确认下载来源、版本、许可证和校验值；自托管项目还需要维护 HTTPS、备份、账户权限与安全更新。
